# Bot Patch Notes v0.3.6

## 1. New Features
- make monster crystals command
- disassemble trash command
- need anything? command now bypass throttles + now also trigger pots/ammo check

## 2. Changes / Improvements

- Bot melee reach now matches the equipped weapon instead of a fixed distance
- Improved bot combat behavior - will now try to reposition/move closer to mob cluster to use AoE skills
- Bots now have 15s looting delay on bot dropped items - you no longer need to race to loot it before other Bots
- Bots now follow same looting rule as players(no looting mob drop by outside party)
- Improved bots pathfinding, taking shorter routes and use portals more effectively
- Small performance improvements

## 3. Bug Fixes

- Fixed bots using an AoE skill (e.g. Slash Blast) on a single monster instead of the stronger single-target skill (e.g. Power Strike)
- Fixed bot being able to downjump in downjump blocked areas
