# Bot Patch Notes v0.3.4

## 1. New Features

- Shadow Partner now doubles bot ranged damage lines (basic and skill) so when the buff lands on a bot (e.g. via a future admin command) Avenger, double/triple throw, Iron Arrow, and basic ranged attacks all benefit automatically.

## 2. Changes / Improvements

- Bot grinding/target selection logic
- Bot now pivots to a closer mob behind it instead of standing idle when its current target is unreachable on the bot's facing side

## 3. Bug Fixes

- Fixed bot attacking mobs that should be out of their range
- Fixed bots casting bulleted ranged skills (Double Shot, Strafe, etc.) without enough ammo
- Fixed bow/crossbow bots firing Power Knockback as a 400 px ranged shot — it now routes as a melee swing with its WZ-defined close-range reach (knockback effect still pending)
- Fixed ammo gate for skills using `bulletConsume` (Avenger 3 stars/cast) — previously the bot only required `bulletCount` (1) and would attempt to fire with too few stars
- Iron Arrow and Avenger now use their measured per-skill vertical reach (Iron Arrow ~30 px above feet, Avenger ~60 px) instead of the generic ±50 px ranged band, so the bot will no longer claim hits on mobs visibly above/below the projectile line
