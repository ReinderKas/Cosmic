# Bot Patch Notes v0.3.4

## 1. New Features

- Shadow Partner skill support
- Power Knockback skill support (dmg only, still no knockback implemented)
- new `trade reserve <page>` command - bot show you what equips they have reserved

## 2. Changes / Improvements

- Bot grinding/target selection logic
- Bot now pivots to a closer mob behind it instead of standing idle when its current target is unreachable on the bot's facing side
- Iron Arrow + Avenger (piercing attack skills) targeting/hitbox improvements
- Better sorting for trading equips

## 3. Bug Fixes

- Fixed bot attacking mobs that should be out of their range
- Fixed bots casting bulleted ranged skills (Double Shot, Strafe, etc.) without enough ammo
- Fixed bow/crossbow bots firing Power Knockback as ranged shot
- Fixed bot sometimes not equipping gears
- Fixed bot sometimes attacking for 1 dmg using nonexisting skill
- Fixed crashing when talking to scripted NPCs
