# Bot Patch Notes v0.3.3

## 1. New Features

- Bots now benefit from mastery skills
- Bots now reacts to nearby people scrolling
- Sell trash command (will still keep gear with +6 or more stats, scrolled, or wep with good attack stats)
- Bots can choose between multiple learned attack skills instead of only one cached single/AoE skill; Dragon Roar only becomes eligible above 50% HP and with either 10+ targets or a nearby healer, then still competes by normal estimated damage
- Optional Bot LLM for chatbot support (crazy demanding and has no gameplay effects)

## 2. Changes / Improvements

- Optimized gear optimizer - significant speedup/reduce stutters on item pickup and bot trade commands
- Improved gear reservation logic - bots should now hoard less items it isn't going to use
- Added more bots lines
- Improved roam mode to stay more local by only expanding to adjacent regions that also have an easy 1-hop return path, avoiding overly broad patrols through one-way drops / ledge walk-offs.
- Improved trade etc command - bot prioritize sending things you already have first, better chance to merge stacks naturally
- Spear/polearm stab-vs-swing damage multipliers now match the randomized attack stance for basic attacks and normal attack skills.

## 3. Bug Fixes

- Fixed bot sometimes not equipping gear
- Fixed bot sometimes equipping gear it shouldn't be able to
- Fixed bot getting stuck in a loop catching/dropping off ropes
- Fixed bot sometimes getting stuck in a trade with each other
- Fixed bot getting stuck "casting" MP Eater/other passive skills
- Fixed bot combat skill selection for Dragon Knight/Spearman skills: Power Crash is no longer treated as damage
- Fixed bot sentry mode not requesting pots+ammo / not rebuffing (sentry/roam/grind now share the same active-mode setup path)
- Fixed "need pots?" / "anyone have hp pots" / "need arrows" triggering duplicate replies and duplicate trade requests when you have multiple bots
- Fixed bot not able to trade from medal/belt slot
- Fixed bot sometimes unable to reach shop
- Fixed "Unable to find 'bot'" errors when trying to spawn bots through buddy menu, reducing spam
- Fixed bot being able to ap reset below legal values (4 dex dexless sin >.<)
- Fixed bot not resetting AP on 1st job advancement
