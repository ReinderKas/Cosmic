# Bot Patch Notes v0.3.3

## 1. New Features

- Bots now benefit from mastery skills
- Bots now reacts to nearby people scrolling
- Optional Bot LLM for chatbot support
- Sell trash command (will still keep gear with +6 or more stats, scrolled, or wep with good attack stats)

## 2. Changes / Improvements

- Optimized gear optimizer - significant speedup/reduce stutters on item pickup and bot trade commands
- Improved gear reservation logic - bots should now hoard less items it isn't going to use
- Added more bots lines

## 3. Bug Fixes

- Fixed bot sometimes not equipping gear
- Fixed bot sometimes equipping gear it shouldn't be able to
- Fixed bot getting stuck in a loop catching/dropping off ropes
- Fixed bot sometimes getting stuck in a trade with each other
- Fixed bot getting stuck "casting" MP Eater/other passive skills
- Fixed bot sentry mode not requesting pots+ammo / not rebuffing (sentry/roam/grind now share the same active-mode setup path)
- Fixed "need pots?" / "anyone have hp pots" / "need arrows" triggering duplicate replies and duplicate trade requests when you have multiple bots
- Fixed bot not able to trade from medal/belt slot
