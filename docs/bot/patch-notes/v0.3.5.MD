# Bot Patch Notes v0.3.5

## 1. New Features

- Per-level EXP rate bonuses: EXP now scales up as you level (x2 up to x6 at higher levels)
- Speed Infusion now works on bots
- Skill buff toggle (workaround for broken skills)

## 2. Changes / Improvements

- More accurate bot attack speed scaling
- Improved bot retreat behavior / fixed bot oscilating in place when swarmed by mobs
- Optimized bot trade/search by item name commands to be faster and non-blocking
- Smarter bot trading: hands over scrolls you already own first, so they stack and save inventory slots

## 3. Bug Fixes

- Fixed bots attacking too quickly
- Fixed bots facing the wrong direction after attacking
- Fixed bots sometimes failing to resupply + will now explain why failed
- Fixed bots infinite debuffing / bot misclassifying some skills
- Fixed rare server crashes when inventories or summons changed mid-update
